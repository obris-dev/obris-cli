# Contributing

Thanks for your interest in contributing to the Obris CLI!

## Setup

```bash
git clone https://github.com/obris-dev/obris-cli.git
cd obris-cli
uv sync
```

Run commands locally:

```bash
uv run obris --help
```

## Submitting changes

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Test locally against your Obris account
4. Open a pull request with a clear description of what and why

## Reporting bugs

Open an issue on [GitHub](https://github.com/obris-dev/obris-cli/issues) with:

- What you expected to happen
- What actually happened
- Your OS and Python version (`python --version`)

## Questions?

Reach out at [support@obris.ai](mailto:support@obris.ai).
